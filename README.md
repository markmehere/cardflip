# Cardflip

#### By Mark Pazolli

The PIXI.js card flip demo for PIXI.js v5 written in TypeScript and 
using ES6 classes (although a recent update to the original did 
this too).

## Run

You must have `python3` installed.

To run:

```
npm install
npm start
```

## Develop

To develop using watch mode:

```
npm install
npm run watch
```
